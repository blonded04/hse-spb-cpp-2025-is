# Общее

Цель занятия: познакомиться со структурами.

# Базовые упражнения (`0x`)

## `01-point`

Реализуйте структуру `point`, двумерную точку с целочисленными координатами в декартовой системе координат.

## `02-point-scale`

Скопируйте структуру `point` из вашего решениея задачи `01-point`. Напишите функцию `point_scale`, которая изменяет точку, домножая её координаты на предоставленный целочисленный коэффициэнт.

## `03-linked-list`

Напишите связный список. Реализуйте методы `list_find` и `list_insert`. При компиляции также линкуйтесь с `one_more_tu.c`, это нужно чтобы убедиться, что вы понимаете концепцию единиц трансляции.

Убедитесь, что у вас не утекает память: запускайте программу с `address-sanitizer` и под `valgrind`.
