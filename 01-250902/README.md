## `03-compile-me`
Научитесь пользоваться компилятором: прочитайте файл `03-compile-me.c`, скомпилируйте его. Вы столкнетесь с ошибкой компиляции, исправьте её. После этого перекомпилируйте код и запустите. Эта программа должна выводить `Hello, world!` на экран.

Пример вызова компилятора `gcc` для компиляции файла `myfile.c`: `gcc myfile.c`. 
Эта команда создаст бинарный исполняемый файл `a.out` в текущей директории.
Чтобы запустить файл `run.me` нужно написать в консоль: `./run.me`

Попробуйте назвать выходной файл от компиляции по-другому (например, `03-compile-me`), для этого передайте при компиляции флаг `-o` (сокращение от "output"), и после этого флага напишите через пробел желаемое название выходного файла.

## `04-compile-us`

## `06-compile-with-warnings`
БольшУю часть ошибок можно отловить на этапе компиляции, например, неаккуратный программист может завести переменную и не использовать её, случайно поделить на ноль, и многие другие ошибки. Компилятор предоставляет warnings: предупреждения на этапе компиляции о том, что в программе (скорее всего) проблема. Предупреждения компилятора помогают сэкономить часы отладки (дебага) программы.

Прочитайте файл `06-compile-with-warnings.c` и скомпилируйте его с предупреждениями. Для этого добавьте компилятору флаги `-Wall -Werror -Wextra`. Исправьте программу (уберите всё лишнее).
