## `05-swap`
Реализуйте функции `swapInt` и `swapFloat`, меняющие значения переданных им переменных.

Важно: в данном примере никаких проблем с floating point arithmetics нет - объясните себе почему. Вспомните почему `0.1 + 0.2 = 0.30000000000000004`? На первой лекции почитать про устройство чисел с плавающей точкой было домашним заданием, которое вас могут спросить на экзамене. На случай если вы не читали:
* RU: [Наглядное объяснение чисел с плавающей запятой](https://habr.com/ru/articles/337260/)
* EN: [Why are floating point numbers inaccurate?](https://stackoverflow.com/questions/21895756/why-are-floating-point-numbers-inaccurate)
* Дополнительное чтение (на английском): [Fast Inverse Square Root — A Quake III Algorithm](https://www.youtube.com/watch?v=p8u_k2LIZyo)

## `07-count-symbols`
Реализуйте функцию `count_symbols(string, symbol)`, вычисляющую количество символов `symbol` в строке `string`.

Решите задачу двумя способами:
* Используя цикл `while` и не пользуясь стандартной библиотекой.
* Используя цикл `for` и функцию `strlen` из стандартной библиотеки.
  * Загуглите как пользоваться функцией `strlen` самостоятельно.
