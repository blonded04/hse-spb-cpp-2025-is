# Общее
Цель занятия: закрепить алгоритмы из стандартной библиотеки (ещё раз). Предлагается активно пользоваться [cppreference](https://en.cppreference.com). 

**На этом занятии запрещается пользоваться циклом `for`, если в задании не указано обратное**.

На этом занятии предлагается написать небольшой набор функций:
```c++
void task(std::vector<int>&);
```

# (`0x`) Базовые упражнения
## `01-increment-each`
Напишите функцию, которая принимает вектор целых чисел, и увеличивает каждое число вектора на единицу.

См. `std::transform`.

## `02-random-vector`
Напишите функцию, которая принимает вектор целых чисел, и заменяет все его элементы на случайные числа.

См. `std::generate`, `std::rand`.

## `03-pivot-modulo-two`
Напишите функцию, которая принимает вектор целых чисел, и переставляет в нем элементы так, что в векторе сначала идут только чётные элементы, а потом только нечётные.

См. `std::sort`.

## `04-unique`
Напишите функцию, которая принимает вектор целых чисел, и оставляет в нём только уникальные элементы, чтобы в векторе не было повторений.

См. `std::sort`, `std::unique`.

## `05-stdout-oneliner`
Напишите функцию, которая принимает вектор целых чисел, и в одну строчку выводит его содержимое в консоль.

См. `std::copy`, `std::ostream_iterator`.

## `06-all-permutations`
Напишите функцию, которая принимает вектор целых чисел, а после выводит в консоль всевозможные перестановки элементов этого вектора.

Нужно пользоваться циклом `while` / `do`.

См. `std::next_permutation`. 

## `07-sum`
Напишите функцию, которая принимает вектор целых чисел, и возвращает сумму его элементов.

См. `std::accumulate`. 

## `08-sum-squared`
Напишите функцию, которая принимает вектор целых чисел, и возвращает сумму квадратов его элементов.

См. `std::accumulate`. 

## `09-scalar-greater-count`
Напишите функцию, которая принимает два вектора целых чисел одинакового размера, и возвращает количество элементов первого массива таких, что они больше соответствующего им элемента второго массива.

См. `std::inner_product`, `std::plus`, `std::greater`.

## `10-read-sort-print-strings`
Напишите функцию, которая считывает вектор строк из stdin, сортирует их, а после выводит на экран в отсортированном виде так, чтобы каждая строчка была на новой строке.

См. `std::istream_iterator`, `std::copy`, `std::back_inserter`, `std::sort`, `std::for_each`.
