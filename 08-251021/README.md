# Базовые упражнения (`0x`)
## `01`
Попросите семинариста рассказать вам про константные указатели и указатели на константы.
```c
const char* s = "Hello";
int a[100]; // int* const
const int b[100]; // const int* const
size_t strlen(const char* s);
```

**Важно**: компилируйте программу с выключенными оптимизациями `-O0`.

## `04-error-code`
Запустите программу `04-error-code` и выведите код её возврата: для этого нужно после запуска программы ввести в консоль `echo $?`, чтобы вывести код возврата последней программы.

Поменяйте `0` в `return 0` на другие числа, посмотрите какой будет код возврата.

Стандартная конвенция: корректно завершающиеся программы должны возвращать `0`, а иначе ненулевое число.

## `05-fscanf`
Воспользуйтесь `fscanf` (для этого предварительно прочитайте про него на C reference). Также почитайте про `fopen` и `fclose`.

Считывайте строчки длинны не более `10` до тех пор, пока они не закончатся в консоли. В процессе работы программы выводите строку, которую вам ввели, а в конце выведите количество успешно считанных строчек.

В случае, если строка слишком большая (больше `10` символов), завершите программу с кодом ошибки `1`. Убедитесь, что даже в случае ошибки, вы закрываете открытые файлы.

* Убедитесь, что вы не получаете UB.
* Научитесь проверять на EOF (end of file, конец файла), то есть что ввода больше не будет.
